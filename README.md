# MathGamesUB

The code is from DogeNetwork's dogeUB

### Production:
Click the green "code" button on the top right of your screen and create a new codespace on main. Inside that codespace, enter the following commands in the terminal:

```bash
npm i           #installs NPM, only needs to be run once
npm run build   #Builds the server, only needs to be run once
node server.js  #Starts the server localy, needs to be run every time you go back into the codespace
````
